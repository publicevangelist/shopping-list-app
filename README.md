# 🛒 쇼핑 리스트 앱

아이템 추가·삭제·체크 기능을 갖춘 심플한 쇼핑 리스트 웹 앱입니다.

## 기능

| 기능 | 설명 |
|------|------|
| 아이템 추가 | 입력 후 Enter 또는 추가 버튼 |
| 체크 | 왼쪽 원형 버튼으로 완료 표시 / 취소 |
| 삭제 | 오른쪽 × 버튼으로 개별 삭제 |
| 필터 | 전체 / 미완료 / 완료 탭 전환 |
| 일괄 삭제 | 완료된 항목 한 번에 삭제 |
| 데이터 유지 | localStorage에 저장 — 새로고침해도 유지 |

## 실행 방법

`shopping-list.html` 파일을 브라우저에서 바로 열면 됩니다. 서버 불필요.

## 자동 테스트 실행

```bash
npm install
npx playwright install chromium
npm test
```

Playwright 기반 테스트로 21개 시나리오를 자동 검증합니다.

## 테스트 항목

- 아이템 추가 (Enter / 버튼 / 빈 입력 무시 / 최신순 정렬)
- 체크 ON·OFF / 취소선 표시 / 통계 업데이트
- 아이템 삭제
- 필터 탭 (전체·미완료·완료)
- 완료 항목 일괄 삭제
- localStorage 새로고침 후 데이터 유지
